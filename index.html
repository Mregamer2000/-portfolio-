Things I've made

VR · Multiplayer · Browser

Void Sector

A multiplayer 3D FPS that lives entirely in one HTML file. Three.js for the rendering, Ably for the real-time sync — no backend, no build step, just open and play.

Unity · VR

VR Smash Bros Fan Game

A physics-driven VR fighting game with PUN2 networking. I wrote the hitbox/knockback systems, the VR rig teleport, and the per-platform respawn logic. Built on the Gorilla Tag SDK.

Blender · Level Design

Ninja Clash VR — Koretsu Map

A map I'm building for Ninja Clash VR. Modeled in Blender with a scripted, coordinate-driven workflow so I can keep the triangle budget tight without losing sight lines or verticality.

Web · Single-file

XP Games OS

A browser desktop styled after Windows XP — chat, screenshare, ambient audio, leaderboards, mini-games, a whiteboard, and admin tooling. Originally built for classroom use.

Web · Realtime

Lumina — Student Hub

A single-file student hub with a dreamcore / CRT look. Real-time collaboration features built on top of Ably.

Unity · Education

Unity C# Learning Platform

An interactive C# learning site with a practice code editor. It talks to the Anthropic API to give real-time code review and concept explanations while you work.

Unity · VR · Horror

Containment Dread

An SCP-inspired VR horror experience for Gorilla Tag. A single FacilityDreadSystem.cs runs the show — dynamic audio, lighting beats, and environmental storytelling.

Three.js · Single-file

FNaF HTML5 Clone

A Five Nights at Freddy's recreation in a single HTML file, plus a low-poly 3D Three.js version with a working camera system and power-management loop.
